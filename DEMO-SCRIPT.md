# 90-second demo — narration + what to click (LIVE Vertex AI version)

Record a screen capture with a phone-size browser window. Speak the **bold** lines; do the _(actions)_ in sync.

**Which mode to record on:**
- **Best — real AI:** run on `http://localhost:8000/` (your Mac, with `VERTEX_PROXY_URL` set). Every diagnosis is genuinely generated live by Gemini 2.5 on Vertex AI. This is the strongest thing to show.
- **Safety net:** if venue Wi-Fi is bad on the day, present the public `?demo=1` link — the flow looks identical. Keep a pre-recorded copy of the live run either way.

## Recording tips
- Do one silent dry run so clicks are smooth and the live call has warmed up.
- Have a real leaf photo saved on the machine before you start.
- If a live call is ever slow or errors mid-take, just re-record — or fall back to `?demo=1`, which looks identical.
- Keep it under ~100 seconds. Upload unlisted to YouTube or Drive; paste the link into the deck's last slide and the submission form.

It's on the mandatory Google stack — naming it (plus Gemini and Firebase) directly scores points on technical implementation.
